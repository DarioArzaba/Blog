## Introducción

Al visitar un sitio web tu navegador envía el URL junto con tu IP a tu ISP, quien mediante sus servidores NAP y DNS transforma el URL en la IP del servidor final. Se abre una conexión entre tu IP y la IP del servidor para que el servidor pueda enviar su respuesta en forma de paquetes. Se denomina Front-End al contenido de los paquetes y Back-End al sistema en el servidor que le permite decidir que paquetes enviar. Si el servidor nunca cambia el contenido de los paquetes decimos que tiene un sitio estático.

### HTML

Iniciamos creando un archivo ejemplo de HTML llamado `basics.html`. Este archivo muestra ciertos componentes básicos de HTML como declarar el documento, incluir meta información, el titulo, texto, negritas, itálicas, links, imágenes, listas, subtítulos, links en imágenes, divs, spans, tablas y formas.
