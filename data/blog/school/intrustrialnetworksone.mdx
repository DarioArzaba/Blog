Communication is almost as important as any other vital resource.

It creates a world without boundaries, global communities and human information networks.

Tenemos servicios en linea, correo, web, archivos o juegos. 

Estos son generados por servidores que dan la informacion a los clientes.

Los clientes le piden al servidor informacion, sitios web, o los emails al servidor de email.

Cada computadora en una red es un HOST o un END DEVICE.

Los dispositivos intermedios en internet son routers que comunican los nodos.

Email Server: Runs email software server, clients use client software.
Web Server: Runs web server software, clients use browsers.
File Server: Stores corporate and user files, client uses diferent devices to access the files.

Existes distintos dispotivios que se pueden agregar a las redes.
