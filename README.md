## Installation

1. Clone the repo
   ```sh
   git clone https://github.com/DarioArzaba/blog
   cd blog
   ```
2. Install and rebuild NPM packages
   ```sh
   npm install
   npm rebuild
   ```
4. Deploy either in `build` mode or `dev`
   ```js
   npm run build
   ```
